# **Image-Captioning**


#### Auto Image Caption Generator

## Usage
A sample model is provided to try out the program
- Place images in **inputs** directory
- Run *Caption_Generator.py*
- Outputs will be saved to **outputs** directory

## Package Dependencies
- scipy
- numpy
- PIL
- matplotlib
- os
- pickle
- keras
- time
- string
- sys
- tensorflow
- nltk
